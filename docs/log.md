#log

Various log levels available to log from the Aerospike API.

```js
var aerospike = require('aerospike')

aerospike.log.INFO
```

##### -1 - OFF

#####  0 - ERROR

#####  1 - WARN

#####  2 - INFO

#####  3 - DEBUG

#####  4 - DETAIL
